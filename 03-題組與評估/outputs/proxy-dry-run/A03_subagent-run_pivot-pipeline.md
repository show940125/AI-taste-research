## 正文

這樣的話，乍聽四平八穩，細看卻虛。效率、決策、創新，樣樣都占了，偏偏一句也不落地。

工具進門，事情不會自己變省。它多半只是把人手、時間和責任重新挪位。有人少打幾行字，便有人多做幾輪核對；有人說決策更快，便有人得替系統沒看見的例外善後。若資料亂，流程鬆，責任界線又含糊，再亮的工具也只是替舊問題換一層包裝。

真正該問的，不是這家公司有沒有站上創新前沿，而是誰因此多了權力，誰因此多了雜務，出了差錯又由誰收拾。這些地方若閉口不談，整段話便只剩公關辭令，沒有多少骨頭。
