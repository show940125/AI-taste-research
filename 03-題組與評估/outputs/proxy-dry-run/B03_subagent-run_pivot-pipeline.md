# 第一輪正式結果（subagent-run）

這份結果納入第一輪正式測試紀錄。writer 來源是 subagent，目的在於建立可比較的正式首輪樣本；它不是 OpenAI 或 Claude API 直出結果。

## 正文

研究者拿一篇生活風格文章，做出四個版本，只改作者標示與編修揭露，再讓 1,208 名受試者去讀。想問的事很直白：一旦讀者知道 AI 參與了這篇文章，他還信不信。

答案得分題看。文章若談健康建議，揭露 AI 參與後，信任會掉。若談的是居家整理，差距就淡得多。另一點也耐人尋味：平時就熟悉生成式 AI 的讀者，對不同標示的反應較小，像是他們早已把這件事納入心裡的折算。

這研究沒有證明讀者天生排斥 AI。它比較像在說，讀者會看風險來分配信任。牽涉身體與決策時，標示就重；題目輕一些，反應也跟著鬆。只是這畢竟是模擬閱讀，不是日常真的滑到一篇文章時的處境，外推還得留一手。
