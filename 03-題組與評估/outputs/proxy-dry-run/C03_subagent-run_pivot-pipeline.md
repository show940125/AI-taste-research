# 第一輪正式結果（subagent-run）

這份結果納入第一輪正式測試紀錄。writer 來源是 subagent，目的在於建立可比較的正式首輪樣本；它不是 OpenAI 或 Claude API 直出結果。

## 正文

政策寫得好看，常是一回事；真正落到人手上，往往又是另一回事。外頭的人先看見的，多半是它宣稱了什麼：程序簡化、責任清楚、效率提高。可做事的人心裡明白，事情哪有這樣乾淨。表格雖薄，後頭卻常壓著一整套沒寫明的門道。什麼情況算例外，資料缺一角時怎麼補，指示不全時由誰扛下去，紙上未必交代，現場卻不能不處理。

於是看起來越簡單的制度，有時反而越仰賴人暗地裡墊工。那些多出來的判斷、加上的時間、臨時補上的手腳，平常不入帳，也不進說明書。可沒有這些東西，制度根本轉不動。

所以評政策若只看它嘴上說了什麼，常會看偏。還得回頭問一句：它到底悄悄把多少工作留給了執行的人。
