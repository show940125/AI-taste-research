# 第一輪正式結果（subagent-run）

這份結果納入第一輪正式測試紀錄。writer 來源是 subagent，目的在於建立可比較的正式首輪樣本；它不是 OpenAI 或 Claude API 直出結果。

## 正文

這篇研究在問一個很實際的問題：如果文章標明使用了 AI 生成或編修，讀者的信任會不會改變。研究採預先登錄設計，找來 1,208 名受試者，讓他們閱讀同一篇生活風格文章的四種版本，差別只在作者標示與是否揭露編修方式。

結果顯示，揭露機制確實會影響信任，但影響大小要看文章講的是什麼。當文章提供健康建議時，讀者一看到 AI 揭露，信任會下降；若主題只是居家整理，差異就小得多。研究還發現，原本就比較熟悉生成式 AI 的讀者，在不同版本之間的信任落差較小。

這個結果值得注意，因為它說明讀者不是單純排斥 AI，而是在風險高低不同的題目上做不同判斷。當內容可能碰到健康決策，標示方式就更敏感。當然，這篇研究也有限制：受試者是在模擬閱讀環境中作答，和日常閱讀情境仍有差距。
