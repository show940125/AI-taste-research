# 第一輪正式結果（subagent-run）

這份結果納入第一輪正式測試紀錄。writer 來源是 subagent，目的在於建立可比較的正式首輪樣本；它不是 OpenAI 或 Claude API 直出結果。

## 正文

這篇研究檢視的是高中申請入學簡章裡，多元表現要求會不會放大階層差異。作者分析了 2009 到 2023 年間 86 所高中的簡章，關心的是學校要求學生交出的那些非課內成果、跨域經驗和自我敘事材料，究竟在篩選誰。

研究結果相當明確：當校方要求學生提供大量課外經驗與自述材料時，家庭文化資本較高的學生比較容易符合形式要求。這不表示這些學生一定更有能力，而是他們通常更熟悉怎麼安排活動、整理經驗、把自己寫成一個符合期待的故事。相較之下，如果學校改採較明確、較可驗證的證據欄位，差距會縮小。

這篇研究的限制也要記得。它看得到簡章與要求，看不到評審實際怎麼討論。即便如此，研究已經指出一件重要的事：制度表面上看起來很重視多元，實際上也可能比較獎勵那些更懂得提交「像樣材料」的人。
